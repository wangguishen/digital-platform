# digital-platform

## 安装依赖包
```
npm install
```

### 运行项目 区别不同环境 不同路径配置
```
npm run serve
```

```
npm run serve:online38
```

```
npm run serve:online:76
```

### 打包项目 区别不同环境 不同打包路径配置
```
npm run build
```

```
npm run build:prod
```

```
npm run build:staging
```

### 运行测试
```
npm run test
```

### 使用eslint进行修补代码
```
npm run lint
```

### 运行单元测试
```
npm run test:unit
```
